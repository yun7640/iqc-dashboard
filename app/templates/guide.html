3. 기본 사용 흐름 (월별 검토 한 바퀴)
  1. 정도관리담당(qc) 로그인 → 대시보드에서 해당 장비·월 현황 확인
  2. '마스터·데이터연동'에서 LIS 파일 업로드(신규 데이터가 있을 때)
  3. 'QC·L-J차트'에서 검사항목별 L-J 차트와 멀티룰 판정 확인
  4. 이탈(위반)이 있으면 '시정조치'에서 CAPA 등록
  5. '월별검토·전자서명'에서 정밀성 종합검토 확인 → 검토의견 작성 → 전자서명(비밀번호 재인증)
  6. 전문의(dr) 로그인 → 검토의견 작성 → 전자서명 → 상태 '검토완료'
  7. '인증 제출용 PDF' 다운로드
4. 데이터 연동 (LIS 업로드)

LIS에서 내려받은 내부QC 파일을 업로드하면 검사항목·Lot·목표값·일별 QC가 자동 등록됩니다(CSV/.xls, 파일명이 깨져도 헤더로 인식).

  • 정밀성(월통계) → 월별검토 정밀성 종합검토
  • L-J 일별 → QC·L-J차트
  • 일일결과 → 물질별 결과
5. QC·L-J차트 (Westgard 멀티룰)

검사항목·레벨 선택 시 L-J 차트(±1/2/3SD)와 일자별 z-score·판정이 표시됩니다.

  • 적용 규칙: 1₂ₛ(경고)·1₃ₛ·2₂ₛ·R₄ₛ·4₁ₛ (10ₓ 미적용)
  • L-J 중심선은 CLSI C24 확립평균, 배정값 편차는 (Mean)%Diff로 별도 평가
6. 월별 검토 · 전자서명

서명은 담당 → 전문의 순서로만 가능하며 각 서명은 비밀번호 재인증을 거칩니다. 서명 시점 데이터 무결성 해시(SHA-256)가 저장되어 변조를 탐지합니다.

  1. 정밀성 종합검토표 확인(전월↔당월 Mean·SD·CV, %Diff, 위반, 적합/확인)
  2. 검토의견 작성(문구 템플릿 선택·삽입) → 저장
  3. 전자서명 → 비밀번호 재입력 → (담당 서명완료)
  4. 전문의 로그인 → 검토의견 → 전자서명 → 검토완료 → PDF 다운로드
7. 시정조치 · 서명 이미지

CAPA: 멀티룰 위반 항목에 원인·조치·재발방지를 기록하고 완료 시 해결 처리합니다.

내 서명: 흰 종이에 서명 후 촬영 → '내 서명' → 파일 선택 → '흰 배경 투명' 체크 → 등록. 전자서명 시 화면·PDF에 표시됩니다.

8. 판정 기준 (적합 / 확인)

아래를 모두 충족하면 적합, 하나라도 초과하면 확인(초과 셀 빨간색 강조):

  • 당월 CV ≤ 그룹 기준(일반 5% / 저농도 10%)
  • |(Mean)%Diff| ≤ 기준(기본 3%)
  • |(CV)Diff| ≤ 기준(기본 3%)
  • 일별 멀티룰 거부(reject) 0건
